Nexus - Face Recognition Surveillance

This project leverages the powerful Flask framework, a micro web framework written in Python, to create a sophisticated application that utilizes advanced computer vision techniques. Its primary aim is to enhance security measures by accurately identifying and tracking individuals in real-time through video feeds. By integrating state-of-the-art algorithms consisting of object detection & tracking alongside machine learning models for face recognition, the system is capable of processing complex visual data efficiently. This approach allows for a robust and dynamic security solution that can be adapted to various environments, offering a high level of precision in monitoring and threat detection. The use of Flask enables rapid development and deployment, as well as creating a suitable user-friendly interface.

Development Tools

  • HTML5
  • CSS3
  • JavaScript & jQuery
  • Bootstrap for styling
  • Flask & Python for back-end

Machine Learning & Data Processing
